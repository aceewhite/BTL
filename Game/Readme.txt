Directions:
Unzip the file and use Codeblocks to run the game . The zip files has all the dll required to run the game. Click on the main.cpp and press run to play the game.

Instructions:
Player Left : Use W and S to control the pad on the left side of the screen
Player Right: Use Arrow Up and Arrow Down to control the pad on the right side of the screen.
The game doesn't have any score limit to determine the winner (not yet) so just have fun and play the game.
